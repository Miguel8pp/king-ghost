Instrucciones para {{ sistema|capitalize }}

Tu archivo ya fue generado. Si no se descargó automáticamente, haz clic aquí para descargarlo.

{% if sistema == 'pc' %}

Pasos en PC (Windows):

  1. Haz clic derecho sobre el archivo PC.bat y selecciona "Ejecutar como administrador".
  2. Se agregarán las entradas al archivo del sistema hosts.
  3. Reinicia tu navegador o el equipo si no ves cambios inmediatamente.
{% elif sistema == 'android' %}

Pasos en Android:

  1. Instala una app de DNS como NextDNS, RethinkDNS o DNSChanger desde Google Play.
  2. Abre la app y busca una opción para cargar configuración personalizada o listas.
  3. Importa el archivo Android.conf desde la carpeta de descargas.
  4. Activa el servicio de DNS personalizado o filtrado según la app.
  5. Reinicia la conexión o dispositivo si no se aplican los cambios de inmediato.
{% elif sistema == 'iphone' %}

Pasos en iPhone:

  1. Descarga el archivo iPhone.mobileconfig.
  2. Abre el archivo desde la app Archivos o desde Safari.
  3. Se abrirán los Ajustes automáticamente para instalar el perfil.
  4. Toca "Instalar", ingresa tu código si se solicita y acepta los permisos.
  5. Verifica que el perfil esté instalado en Ajustes > General > Perfiles.
{% endif %}
